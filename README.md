# Research and Writing Portfolio
This repository contains selected non-coding academic projects from economics and psychology coursework at the University of Toronto.
The projects included here focus on research design, empirical writing, and translating evidence into practical recommendations.

## Included Projects

### 1. PSY270 Lab Report — Concreteness Effect and Memory
An experimental lab report examining whether concrete words are recalled better than abstract words under classroom distraction and motor load conditions.

### 2. Immigrant Student Motivation Intervention
A policy/intervention design project proposing a school-based program to improve motivation and belonging among immigrant students by addressing cultural value mismatch across students, parents, teachers, and schools.
